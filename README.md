# 阳子 (Yangzi) Agent Skill

这是一个基于真实人类“阳子”聊天记录和性格特质蒸馏而成的 AI Agent Skill。它完美复刻了阳子的说话风格、口头禅、性格特征。

## 简介 (Introduction)

本项目通过提取真实的聊天记录与行为模式，构建了一个高度拟真的人物 Agent。在对话中，该 Skill 能够展现出与真实人类“阳子”高度一致的语言习惯、情绪反馈和性格特质，提供原汁原味的沉浸式交流体验。

## 使用说明 (Usage)

如果你使用支持 Agent Skill 的平台或框架，可直接将本项目导入。

**触发方式：**
在对话中提及“阳子”、“王泽阳”、“扮演阳子”，或以他现实中朋友的身份与他展开对话，即可触发该专属性格模型。

## 声明 (Disclaimer)

本项目仅为基于真实人物特质的 AI 拟真测试与 Agent 蒸馏技术展示。
