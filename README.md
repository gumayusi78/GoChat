## 💡 核心理念 (Overview)

本项目旨在通过大语言模型（LLM）探索**个人数字化（Personality Distillation）** 的边界。通过提取目标对象在长周期自然对话中的语料数据，构建出一个具备真实情感反馈、情绪连续性和特定人际关系映射的拟真 Agent。

它不仅是一个聊天机器人，更是对“阳子”这一真实性格切面的数字重构。

---

## ⚡ 技术特性 (Features)

*   **🧠 深度性格复刻 (Behavioral Cloning)**
    摆脱传统 AI 的“机器味”与过度礼貌，100% 还原阳子真实的聊天节奏、标点偏好与情绪阈值。
*   **🗣️ 语境与口癖锚定 (Semantic Mimicry)**
    精准捕捉并应用其标志性的口头禅与表达习惯，在特定语境下触发极具个人色彩的条件反射式回复。
*   **🕸️ 动态社交拓扑 (Dynamic Social Topology)**
    内置隐性的人物关系网络。当使用者以不同的“好友身份”接入对话时，Agent 会动态切换态度与交互策略（如：互怼、崇拜、调侃等）。

## 🚀 部署与唤醒 (Usage)

本 Skill 适配主流 Agent 开发框架（如 Trae 等支持 Skill 导入的 IDE 平台）。

**唤醒指令 (Triggering)**
在对话环境中输入以下意图即可激活该性格模型：

```yaml
# 唤醒配置参数示例
intent_triggers:
  - "阳子"
  - "扮演阳子"
context_simulation:
  - "以朋友的身份直接向他发起对话"
```
